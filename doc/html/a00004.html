UART String Receive Handler

The UART String Receive handler which gets a string sent to the SoC through an unsigned character pointer checks if the string is either START or STOP so that the scanning of the advertisements can be started or stopped respectively.

The scan window timer is a single shot timer with an interval equal to the scan window. At the end of the scan window this function performs 3 different tasks as shown below.

Radio Interrupt Routine

The radio peripheral is configured such that the end of a packet and the end of Received Signal Strength (RSSI) measurement trigger the interrupt. At the end of a packet, it is collected if there is space in the buffer. At the end of the RSSI measurement the RSSI value is stored.
