Warning
Works only for input less than 512000 milli-seconds, or 8.5 min without when RTC_PRESCALER is 0
Parameters
idID of the timer to be used from timer_num
modeMode of the timer as specified in timer_mode
ticksThe number of ticks of the RTC after which the timer expires
handlerPointer to a function which needs to be called on the expiry of the timer
Note
Starting an already started will restart the timer with the current number of ticks passed. is_ms_timer_on can be used to check if a timer is already running.
